🖼️ Media Manager

Simple example of using the Media Manager system for image upload and storage via GitHub.

📤 Drag & Drop

Upload images by dragging and dropping

⚡ Optimized

Auto resize and compression

🔍 Search

Find images instantly

🌐 CDN

Global CDN delivery

1 Authenticate with GitHub

Click the button below to authenticate with GitHub. This is required for uploading images.

2 Open Media Library

After authentication, open the media library to upload and manage images.

3 Use in Your Code

Here's how to use the Media Manager in your own projects:

// Initialize Media Manager const mediaManager = new MediaManager(githubAuth, { assetsPath: 'cms/assets/images', indexFile: 'cms/assets-index.json', maxWidth: 1920, quality: 0.85 }); await mediaManager.init(); // Upload an image const file = fileInput.files[0]; const metadata = await mediaManager.uploadImage(file); console.log('CDN URL:', metadata.cdnUrl); // List all images const images = mediaManager.listImages(); // Search images const results = mediaManager.searchImages('profile'); // Delete an image await mediaManager.deleteImage('img-id');
4 Use CDN URLs in Your HTML

All uploaded images get automatic CDN URLs for fast loading:

<!-- Use the CDN URL from metadata --> <img src="https://cdn.jsdelivr.net/gh/username/repo@main/cms/assets/images/image.webp" alt="My image" loading="lazy">
