🎨 Designer Mode Test Page

Quick Start

  1. Open the main website: index.html
  2. Press Ctrl + Shift + E
  3. Click any highlighted text to edit
  4. Click outside to auto-save

Test Checklist

  • ✅ Designer mode activates with keyboard shortcut
  • ✅ Admin panel appears in top-right corner
  • ✅ Editable elements are highlighted
  • ✅ Text can be edited by clicking
  • ✅ Changes auto-save on blur
  • ✅ "Saved" indicator appears
  • ✅ Version history works
  • ✅ Export/Import functionality works
  • ✅ Changes persist after page refresh

Console Commands

Open browser console (F12) and try:

// Check if designer mode is loaded console.log('Designer Mode:', window.designerMode); // Toggle designer mode programmatically window.designerMode.toggle(); // Check current content console.log('Current Content:', window.designerMode.currentContent); // View version history console.log('History:', window.designerMode.versionHistory); // Manually save window.designerMode.saveAll();

Storage Check

Open browser DevTools → Application → Local Storage and look for:

  • loonix-cms-content - Current content
  • loonix-cms-history - Version history

Troubleshooting

If designer mode doesn't activate:

  1. Open browser console (F12) for errors
  2. Check that designer-mode.js is loaded
  3. Verify content-schema.json exists
  4. Try refreshing the page
  5. Clear localStorage and try again
